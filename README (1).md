## ✅ SECTION-WISE STRATEGY

### **Section A (14 marks) - Speed is Key! ⚡**
- **Time Allocation:** 20-25 minutes max
- **Answer Length:** 3-4 lines per question
- **Format:** Definition + 1 key point + Example (if needed)
- **Tip:** Write in bullet points if allowed, saves time

**Example:**
> **Q: What is Varnashrama System?**  
> A: The Varnashrama system divides society into four varnas (Brahmin, Kshatriya, Vaishya, Shudra) based on occupation and four ashramas (Brahmacharya, Grihastha, Vanaprastha, Sannyasa) based on life stages. It provided social organization in ancient India.

### **Section B (21 marks) - Choose Wisely! 🎯**
- **Time Allocation:** 50-60 minutes (20 min per question)
- **Answer Length:** 1.5-2 pages per question
- **Pick questions where you can write:**
  - Clear introduction (2-3 lines)
  - 5-7 main points with examples
  - Comparison table (if applicable)
  - Conclusion (2-3 lines)

**Pro Tip:** Always pick questions from different units to show breadth!

### **Section C (35 marks) - Maximum Marks! 💪**
- **Time Allocation:** 80-90 minutes (15-17 min per question)
- **Answer Length:** 1-1.5 pages per question
- **Structure:**
  - Introduction
  - 4-6 detailed points
  - Examples/sub-points
  - Conclusion

**Unit Distribution:** Try to cover all 5 units across your 5 answers
